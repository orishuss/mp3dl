mp3dl allows you to quickly and conveniently download mp3 files from youtube, using the successful youtube-dl library.

Dependencies:
    -   Have youtube-dl in your path.
    -   Have a library converting video to audio that youtube-dl supports (avconv / ffmpeg)

Configuration:
    You can edit "config.py" file for configuring the script.

Usage:
    Simply run mp3dl.py.
    A window asking for a list of songs will be opened. Enter names, save & exit.
    Downloading process will begin. When it's done, the output folder will be opened.
